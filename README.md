# 🚀 Projeto Spring Boot + PostgreSQL

Este projeto foi desenvolvido durante um curso na plataforma **Udemy**, com o objetivo de consolidar conceitos fundamentais do desenvolvimento de APIs REST utilizando **Java** e **Spring Boot**.

## 📚 Conceitos Aplicados

Durante o desenvolvimento do projeto foram explorados diversos conceitos importantes, como:

- ✅ Injeção de Dependências (Dependency Injection)
- ✅ Arquitetura em camadas
  - Controller
  - Service
  - Repository
  - Model
- ✅ Spring Security para autenticação e autorização
- ✅ Controle de acesso baseado em perfis (Roles)
- ✅ Lombok para redução de código boilerplate
- ✅ Spring Data JPA (JPA Repository)
- ✅ Validação de dados com Spring Validation
- ✅ Implementação de regras de negócio para restringir o acesso de colaboradores apenas às funcionalidades necessárias

## 🛠️ Tecnologias Utilizadas

- Java
- Spring Boot
- Spring Security
- Spring Data JPA
- PostgreSQL
- Lombok
- Maven

## 🎯 Objetivo

O principal objetivo deste projeto foi aprofundar os conhecimentos em desenvolvimento backend com Spring Boot, aplicando boas práticas de arquitetura, organização do código, segurança e persistência de dados em um cenário próximo ao de aplicações reais.

## 📌 Aprendizados

Ao final do projeto foi possível adquirir experiência prática em:

- Estruturação de aplicações Spring Boot.
- Criação de APIs REST.
- Integração com banco de dados PostgreSQL.
- Implementação de autenticação e autorização.
- Organização do código seguindo o padrão em camadas.
- Aplicação de validações e regras de negócio.
